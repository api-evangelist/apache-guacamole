# Apache Guacamole (apache-guacamole)
Apache Guacamole is a clientless remote desktop gateway that supports standard protocols like VNC, RDP, and SSH. It requires no plugins or client software and provides access to remote desktops through a web browser with a comprehensive REST API for management.

## APIs

### Apache Guacamole REST API
The Guacamole REST API provides endpoints for managing connections, users, groups, sharing profiles, and authentication, along with a JavaScript client API for embedding remote desktop access and an extension API for custom authentication providers.

**Human URL:** [https://guacamole.apache.org/doc/gug/guacamole-rest-api.html](https://guacamole.apache.org/doc/gug/guacamole-rest-api.html)
